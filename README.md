## Fetch Rewards Coding Exercise - Analytics Engineer

This project is a coding exercise for the Analytics Engineer role at Fetch Rewards. It demonstrates how to preprocess unstructured JSON data, build a structured relational database, and analyze data quality issues â€“ all within a [Jupyter Notebook](fetch_query.ipynb).

### Setup and Installation

0. **Prerequisites:**  
   - Python 3.13.2
   - MySQL Server (ensure you have the relevant credentials)
   - JSON data files (`users.json`, `brands.json`, `receipts.json`) in the project directory


1. **Python Environment Setup:**  
   Create and activate your virtual environment:
   ```bash
   python3 -m venv myenv
   source myenv/bin/activate
   pip install -r requirement.txt
   ```
